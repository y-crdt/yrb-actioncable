# yrb-actioncable

> An ActionCable companion for Y.js clients

## Installation

Add this line to your application's Gemfile:

```ruby
gem "yrb-actioncable"
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install yrb-actioncable
```
